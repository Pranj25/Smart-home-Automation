🏠 Smart Home Automation System

📌 Project Overview
The Smart Home Automation System is an IoT-based solution designed to control and monitor home appliances remotely using a Raspberry Pi.
This system provides:
💡 Smart Light Control
🔐 Smart Security System (Face Detection + Alerts)
🌡 Temperature & Humidity Monitoring
🎙 Voice-Based Control
🌐 Remote Access via Web Dashboard
The project integrates hardware, networking, and software to create a fully automated and secure home environment.

🚀 Features
1️⃣ Smart Light Control
Control lights using:
Web Dashboard
Voice Commands
Relay-based switching
Real-time ON/OFF status updates
2️⃣ Smart Security System
Face detection using Python & OpenCV
Intruder detection alerts
Camera-based monitoring
Email/Notification alerts
3️⃣ Temperature & Humidity Monitoring
DHT11/DHT22 sensor integration
Real-time data display
Threshold-based alerts
4️⃣ Voice Detection System
Microphone-based voice input
Python speech recognition
Trigger-based appliance control

🛠️ Technologies Used
💻 Hardware

Jumper Wires & Breadboard
🧠 Software
Python

HTML, CSS

🏗️ System Architecture
Sensors collect environmental data.
Raspberry Pi processes data.
Flask server hosts web dashboard.
Relay module controls appliances.
Camera monitors security.
User accesses system remotely via browser.

🔐 Security Implementation

Secure local hosting
